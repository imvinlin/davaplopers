# Frontend
 
Built with Angular + Tailwind CSS.
 
---
 
## Setup
 
### 1) Install Node.js
If you don't have Node installed:
```bash
brew install node
```
 
### 2) Install Angular CLI
```bash
npm install -g @angular/cli
```
 
### 3) Install dependencies
```bash
cd frontend
npm install
```
 
### 4) Start the dev server
```bash
ng serve
```
